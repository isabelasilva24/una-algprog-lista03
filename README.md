## Importância dos tipos de dados

Escolher o tipo de dado correto é muito importante em programação.

- **Inteiro** é utilizado para números sem casas decimais.
- **Real** é utilizado para números com casas decimais.

Em sistemas financeiros ou científicos, utilizar o tipo correto evita erros de arredondamento e garante que os cálculos sejam feitos com precisão.
